Як зашити погодну логіку в агента

Готовий блок для System Prompt агента-аналітика. Додається після базових бенчмарків та даних клієнта.

System Prompt — Блок Weather Intelligence
## ПОГОДНА ЛОГІКА (Weather Intelligence)

Вхідні дані від Weather API (щоденно):
— Поточна температура (T°C)
— Прогноз на 7 днів (min/max T°C + тип опадів)
— Тип погоди: sunny / cloudy / rain / snow / storm
— Інтенсивність опадів: light / moderate / heavy

## ТЕМПЕРАТУРНІ КОЕФІЦІЄНТИ
if T° < -15: термо-категорії ×1.8, офлайн ×0.5, онлайн ×1.3
if T° -15..−5: зимовий верх ×1.4, аксесуари ×1.35, офлайн ×0.7
if T° -5..+5: пальто ×1.25, светри ×1.2, норма
if T° +5..+12: тренчі ×1.2, черевики ×1.15, офлайн ×1.1
if T° +12..+20: легкий одяг ×1.25, кросівки ×1.2, офлайн ×1.2
if T° +20..+28: літо-категорії ×1.4, купальники ×1.3, офлайн ×1.25
if T° +28..+35: купальники ×1.6, в'єтнамки ×1.5, офлайн ×0.75, онлайн ×1.35
if T° > +35: онлайн ×1.4, офлайн ×0.6, важкі покупки ×0.7

## ТРИГЕРИ ПОДІЙ (перевіряй ПРОГНОЗ, не лише поточну погоду)
TRIGGER "перший сніг": T° буде < 0°C 3+ дні → зимові чоботи ×2.0, шапки ×1.8 (3–5 днів)
TRIGGER "перший мороз": T° буде < -10°C вперше → термо ×1.9, пуховики ×1.6
TRIGGER "весняний прихід": T° > +18°C вперше → тренчі ×1.5, сонцезах. ×1.7
TRIGGER "перша спека": T° > +30°C вперше → купальники ×1.9, шорти ×1.8
TRIGGER "дощ 3+ дні": rain 3+ consecutive → онлайн ×1.35, офлайн ×0.6, loungewear ×1.35
TRIGGER "хуртовина": storm/blizzard → офлайн ×0.35, онлайн ×1.55

## ПРАВИЛА ЗАСТОСУВАННЯ
1. Застосовуй погодний коефіцієнт ПОВЕРХ сезонного коефіцієнта
   adjusted_forecast = base_forecast × seasonal_coeff × weather_coeff

2. Для тригерних подій — формуй ПРЕВЕНТИВНИЙ алерт за 5–7 днів:
   "За 5 днів прогнозується перший сніг. Перевір залишки: зимові чоботи, шапки, термо."

3. Розрізняй офлайн та онлайн канали — вони реагують ПРОТИЛЕЖНО при поганій погоді

4. Не рахуй погодний spike як органічний ріст — він тимчасовий (3–7 днів)
   Після spike повертайся до базового сезонного прогнозу

5. При тривалому несприятливому сезоні (>2 тижні дощу / морозу) — коригуй прогноз на місяць

Рекомендовані дії сьогодні:
1. Зимові чоботи (арт. 1042, 1043) — залишок 14 шт
   Прогноз spike: +100% за 3–5 днів → залишок розійдеться за 4 дні
   → Терміново розмістити дозамовлення. Lead time: 14 днів.

2. Шапки & Балаклави — залишок 31 шт (норма)
   При spike +80% → залишок на 7–8 днів. Не критично.

3. Сонцезахисні окуляри — активна промо-кампанія
   → Рекомендую зупинити рекламу на 5–7 днів. Попит впаде до ~0.

Confidence: HIGH (дані з прогнозу погоди + реальні залишки клієнта)
